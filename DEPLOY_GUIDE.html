Prerequisites — Install These First

// Install Node.js

Download and install from nodejs.org — get the LTS version (v20+).

# Verify Node is installed node --version # should print v20.x.x or higher npm --version # should print 10.x.x

// Install Git

Download from git-scm.com. Required for deploying to Vercel via GitHub.

git --version # verify git is installed
2

Local Setup — Run in 3 Commands

2.1
Create the project folder anywhere on your computer — e.g., Desktop or Documents. Name the folder vimlaji. Copy all the files inside it exactly matching the structure above.
2.2
Open your terminal (Command Prompt on Windows, Terminal on Mac). Navigate into the project folder.
# Navigate to your project folder cd Desktop/vimlaji # adjust path as needed # Install all dependencies npm install # Start development server npm run dev
✅ After running npm run dev, open http://localhost:5173 in your browser. You should see the Vimlaji website running locally!

4

Push to GitHub Free

4.1
Go to github.com → Sign up / Log in → Click New Repository → Name it vimlaji → Set to Public → Click Create Repository.
# Inside your vimlaji folder in terminal: git init git add . git commit -m "Initial Vimlaji website" git branch -M main git remote add origin https://github.com/YOUR_USERNAME/vimlaji.git git push -u origin main
5

Deploy on Vercel — 100% Free Free Forever

5.1
Go to vercel.com → Sign up with your GitHub account.
5.2
Click "Add New Project" → Import your vimlaji GitHub repository.
5.3
Vercel auto-detects Vite. All settings are correct automatically. Just click "Deploy".
5.4
Wait ~60 seconds. Your site is now live at vimlaji.vercel.app 🎉
✅ Every time you push code changes to GitHub with git push, Vercel automatically re-deploys your site in under 1 minute!
6

Connect vimlaji.com Custom Domain

6.1
In Vercel dashboard → Your Project → Settings → Domains → Type vimlaji.com → Click Add.
6.2
Vercel shows you DNS records to add. You need to go to your domain registrar (GoDaddy, Namecheap, Google Domains, etc.) and add these records.
# Add these DNS records at your domain registrar: # For vimlaji.com (root domain) Type: A Name: @ Value: 76.76.21.21 # For www.vimlaji.com Type: CNAME Name: www Value: cname.vercel-dns.com
6.3
DNS changes take 5 minutes to 24 hours to propagate. Vercel automatically adds a free SSL certificate (HTTPS) once verified.
💡 Tip: Vercel's exact DNS values may differ slightly. Always copy them directly from your Vercel dashboard → Domains page. The values above are typical but use Vercel's exact values.
7

How to Update the Website Later

# Make your changes in the code files # Then in terminal inside the vimlaji folder: git add . git commit -m "Update: describe your change" git push # Vercel auto-deploys in ~60 seconds. That's it!
